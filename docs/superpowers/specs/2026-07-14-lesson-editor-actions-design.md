# Lesson Editor Actions Design

## Цель

Кнопки управления уроком не должны перекрывать поля редактора при открытии мобильной клавиатуры.

## Решение

- Удалить дублирующую кнопку «Закрыть»: возврат выполняется стрелкой в шапке экрана.
- Поместить действия сохранения и удаления внутрь конца прокручиваемого содержимого формы.
- Убрать отдельную закреплённую панель, её фон, размытие и верхнюю границу.
- Для нового урока показывать одну полноширинную кнопку «Сохранить урок».
- Для существующего урока показывать удаление и сохранение в нижнем блоке формы, не перекрывающем поля.
- Сохранить существующие обработчики, состояния загрузки и бизнес-логику.

## Проверка

- Регрессионный тест проверяет отсутствие кнопки «Закрыть» и нахождение действий внутри прокручиваемой области.
- Проверка типов, полный web test suite и production build.
- Визуальная проверка мобильного экрана с открытой клавиатурой.
