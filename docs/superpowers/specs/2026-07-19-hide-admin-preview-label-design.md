# Скрытие подписи предпросмотра ролей

## Цель

Убрать видимую надпись «Вид как» из шапки админки, сохранив переключатель режимов предпросмотра.

## Решение

- Удалить только текстовый `span` с подписью «Вид как».
- Сохранить `aria-label="Вид как"` на группе для доступности.
- Не менять варианты режима, права владельца и обработчик переключения.
- Не менять общую геометрию переключателя сверх исчезновения строки подписи.

## Проверка и релиз

- Регрессионный тест подтверждает отсутствие видимой подписи и наличие доступного названия.
- Выполняются типы, тесты, production-сборка и мобильная проверка админки.
- Изменение публикуется следующей версией с записью в журнале обновлений.
