# Визуальное состояние открытого раздела клиента

## Цель

Сделать открытый сворачиваемый раздел в карточке клиента заметно, но ненавязчиво отличимым от закрытых разделов во всех темах приложения.

## Решение

- Сохраняем существующие размеры, отступы, типографику и механику элемента `details`.
- Для `.admin-client-compact-section[open]` используем более заметную рамку, смешанную из семантического `--accent` и текущего `--border`.
- Для шапки открытого раздела добавляем лёгкую тонировку на основе `--accent-soft` и текущей поверхности.
- Стрелка открытого раздела получает цвет `--accent`; существующий поворот сохраняется.
- Состояние закрытых разделов не изменяется.

## Темы и доступность

Все оттенки строятся только из семантических переменных темы, поэтому решение работает в светлых и тёмных палитрах. Различие не опирается только на цвет: открытое состояние также сохраняет повёрнутую стрелку и раскрытое содержимое.

## Проверка

- Регрессионный тест подтверждает наличие отдельного стиля `[open]` для рамки, шапки и стрелки.
- Полный набор тестов и production-сборка должны завершиться успешно.
- Высота закрытой строки остаётся 44 px, а центрирование не меняется.
