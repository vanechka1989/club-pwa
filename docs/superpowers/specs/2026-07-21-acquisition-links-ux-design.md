# Улучшение рекламных ссылок

## Интерфейс

- В карточке клиента значение источника находится справа в шапке «Источник клиента»; отдельная вложенная карточка источника удаляется.
- UTM-метки располагаются сразу под шапкой и показываются только когда значение заполнено.
- Кнопка «Метки и ссылки» располагается в начале экрана рекламной аналитики, до KPI и графиков.
- В генераторе название ссылки обязательно. Из `utm_source`, `utm_medium`, `utm_campaign`, `utm_content` должно быть заполнено хотя бы одно поле.
- Пустые UTM-параметры не добавляются в готовый URL.
- Список ссылок сортируется по дате создания: новая ссылка всегда первая.
- Карточка ссылки показывает автора и дату создания. Для старых системных ссылок без автора отображается «Система».

## Совместимость

Миграция базы не требуется: существующие строковые колонки принимают пустые строки, а `created_by_user_id` уже сохраняется. API расширяется nullable-полем автора.

## Проверка

Контрактные, серверные и UI-тесты проверяют минимум одну UTM-метку, отсутствие пустых параметров, автора, сортировку и расположение элементов. Мобильная проверка выполняется на ширинах 320–1440 px.
