# Прямое удаление карточки урока

## Цель

Администратор может удалить карточку урока прямо из раскрытого модуля, не открывая редактор карточки.

## Интерфейс

- Между кнопками изменения порядка каждой карточки размещается кнопка с иконкой корзины.
- Все три кнопки имеют одинаковую геометрию и минимальную область нажатия 44 × 44 px.
- Корзина использует семантический цвет опасного действия, но не перекрывает обложку карточки.
- Кнопка видна только пользователям с правом управления модулями.
- Для горизонтальных карточек используется тот же порядок: предыдущая позиция, удалить, следующая позиция.

## Поведение

1. Нажатие на корзину не открывает карточку.
2. Появляется единое окно приложения с названием урока и предупреждением о хранении в удалённых 7 дней.
3. Отмена ничего не меняет.
4. Подтверждение вызывает существующий API удаления, удаляет карточку из модуля и добавляет её в системный раздел «Удалённый контент».
5. При ошибке карточка остаётся на месте и показывается существующее сообщение об ошибке.
6. Удаление из редактора урока продолжает работать через тот же общий обработчик.

## Проверка

- Компонентный тест проверяет наличие корзины между стрелками.
- Тест проверяет отмену без удаления и подтверждение с переносом в удалённые.
- Проверяется отсутствие открытия редактора при нажатии на корзину.
- Выполняются типизация, полный набор тестов и production-сборка.
