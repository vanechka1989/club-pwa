# Действия клиента перед источником

В карточке клиента после краткой сводки показывать панель «Действие». Кнопку «Написать клиенту» оставить между двумя блоками. Карточку «Источник клиента» перенести ниже кнопки и предупреждений, перед разделом активности.

Меняется только порядок отображения. Права доступа, обработчики действий, сообщения и UTM-данные остаются без изменений.
