# Удаление дублирующей сводки привлечения

## Решение

На экране «Рекламные ссылки» оставить четыре верхние KPI-карточки: визиты, регистрации, оплатившие и выручка. Удалить расположенный ниже блок «Статистика переходов», потому что он повторяет визиты, регистрации и оплаты без новой информации.

После KPI-карточек сразу показывать интерактивный блок «Динамика», затем источники. Расчёты, API и переходы в детализацию по дням не менять.

## Проверка

- В компоненте остаётся единственная сводка KPI.
- Текст «Статистика переходов» и связанные элементы воронки отсутствуют.
- «Динамика» располагается после KPI.
- На мобильных ширинах не появляется пустой контейнер или лишний отступ.
