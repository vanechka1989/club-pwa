# Понятные подписи UTM-полей

## Цель

В форме создания рекламной ссылки должно быть сразу понятно, какое назначение у каждого поля и как называется соответствующий UTM-параметр.

## Решение

Каждое из четырёх полей получает двухуровневую подпись:

- «Источник трафика» — `utm_source`;
- «Тип канала» — `utm_medium`;
- «Название кампании» — `utm_campaign`;
- «Объявление / вариант» — `utm_content`.

Понятное название остаётся основным, техническое имя выводится меньшим акцентным текстом. Значения, API и правила валидации не меняются.

## Адаптивность и проверка

Подписи не должны выходить за границы двухколоночной формы на ширине 360–390 px. На ширине до 359 px сохраняется существующий переход в одну колонку. Автоматический тест фиксирует все четыре пары подписей; затем выполняются проверка типов, полная сборка и тесты.
